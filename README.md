# My Portfolio

A modern and responsive personal portfolio website built using React.js, Tailwind CSS, and Framer Motion.

## 🚀 Features

- Responsive Design
- Modern UI
- Resume Viewer
- Skills Showcase
- Project Showcase
- Education Section
- Contact Information
- Smooth Animations

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Framer Motion
- Vite
- Git & GitHub
